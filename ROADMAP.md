# BulletLab Roadmap

This roadmap reflects the long-term vision for BulletLab. It outlines the planned features and architectural directions for the framework.

> **Note:** These roadmap items represent goals and planned directions rather than strict promises or guaranteed release dates. Development is community-driven and priorities may shift based on user feedback.

## Current (Completed)

- [x] Robot object API
- [x] Joint API
- [x] Link API
- [x] ImGui UI
- [x] Telemetry
- [x] Experiment API
- [x] Documentation
- [x] Cookbook
- [x] AI snippets
- [x] Community robot ecosystem (BulletLab Arsenal)
- [x] Camera improvements (Dynamic tracking)
- [x] Robot inspector
